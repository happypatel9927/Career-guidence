Career Guidance Platform (Eduvators)

A smart Career Guidance Website that helps students make informed career decisions based on their interests, education background, and skills.
The platform uses AI-powered recommendations, quizzes, and a multilingual chatbot to guide users toward suitable career paths.

 Features:

 User Authentication

 Secure Sign Up & Login
 Login using Email / Google (Gmail)
 Personalized user profiles

 Career Assessment Quiz

 Interest-based and education-based questions
 Easy multiple-choice format
 Helps understand user strengths and preferences

 AI-Based Career Recommendations

 Career suggestions generated based on quiz responses
 Recommendations include:

 Suitable career paths
 Required skills
 Learning roadmap

 AI Chatbot (Eduvators Bot)

 Integrated chatbot for career guidance
 Answers user queries related to:

 Careers
 Courses
 Skills
 Exams
 Supports **multiple Indian languages**
 Shows the same recommendations inside the chatbot

 Multilingual Support

 Career guidance available in **Indian regional languages**
 Makes the platform accessible to more students

 Responsive Design

 Works smoothly on **desktop, tablet, and mobile**
 Clean and user-friendly UI
